# API 中转站简介

「中转站」（也叫 API 中转 / relay）是指转售大模型 API 访问能力的第三方服务：它们在上游对接 OpenAI、Anthropic、Google 等官方 API（或各种渠道的额度），对下游用户提供一个兼容官方格式的接口地址和自己发放的 API Key。

## 为什么会有人用中转站

- **支付门槛**：官方 API 通常需要外币信用卡，中转站一般支持支付宝/微信充值
- **网络门槛**：部分官方服务在大陆无法直连，中转站的接入点通常可以直连
- **价格**：部分中转站价格低于官方定价（低价的来源渠道不一，见下文风险）
- **聚合**：一个 Key 同时用多家模型，方便在 Claude Code / Codex 等工具间共用（配合 [cc-switch](./tech-cc-switch.md) 切换更方便）

## 使用方式（通用流程）

1. 在中转站注册、充值，获取 API Key 和 Base URL
2. 在工具中把接口地址指向中转站，例如 Claude Code 设置 `ANTHROPIC_BASE_URL` 和 `ANTHROPIC_API_KEY`
3. 正常使用；用量在中转站后台查询

## 风险与注意事项

使用中转站前务必了解这些风险，**自行判断、量力而为**：

- **隐私风险**：你的全部请求内容（代码、文档、对话）都会经过中转站服务器，敏感内容不要发
- **资金风险**：预充值模式下，站点关停后余额通常无法追回；不要一次充太多
- **稳定性**：高峰期限速、排队、实际模型与标注不一致都可能出现
- **合规风险**：低价渠道可能来自违反上游服务条款的方式（共享账号、盗刷等），使用此类渠道的账号可能被上游封禁
- **质量差异**：同名模型在不同渠道可能有上下文截断、模型替换等情况，可用固定问题自行验证

## 如何辨别相对靠谱的中转站

- 运营时间长、有公开社区和口碑记录
- 价格接近官方定价（过低价格需要谨慎）
- 支持小额充值、按量计费透明
- 明确标注模型来源和倍率
