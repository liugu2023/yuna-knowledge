# 校园网简介

同学们通过宿舍网线、路由器，教学区的公共 Wi-Fi 上网时，都是通过学校的`校园网`系统接入互联网。  
`校园网`相比流量的优势在于：  

- 速度相对稳定，不受信号干扰
- 选课时更快
- 可以访问一些学校专属资源，如知网、正版化激活等。

下面介绍一些关于校园网的简单问题。

## 如何接入

宿舍网线，学校教学楼区域的WiFi信号`iYanDa`，学校宿舍区域的WiFi信号`iYanDa-Dormitory`，学区有线、无线网络均采用校园网认证系统，详情请看[连接与认证方式](https://www.bing.com)一文。

## 要不要办理校园流量卡（校园卡）业务？

> 💡 **提示**
> **本文对同学们办卡与否无任何有意或无意的暗示。考虑到校园卡资费较高，同学们应根据个人需求和经济能力合理作出决定。**

学校为同学们每人每月提供60GB的免费校园网流量，同时对[B站](https://www.bilibili.com)和[魔搭社区](https://www.modelscope.cn)两个网站免流量开放。校园网免费流量不可加购，**不可结转至下月**。通过这一免费套餐，同一个账号可以同时登录**三台**设备的校园网，在全校所有宿舍、教学楼、图书馆等校园网覆盖区域均可使用；对于手机流量足够，平时无刷视频、游戏等大量联网需求的同学已经足够。  

但如果刷视频、游戏和下载等流量消耗需求较大的，可以考虑办理校园卡。大多套餐为校园卡包月套餐，校园卡本体提供每月25GB的国内通用流量和280GB的河北省省内流量，同时赠送校园宽带服务，在校园网认证时选择对应运营商即可使用，使用此宽带时无限流量，但同时只能连接**一台**设备。（*but ！= 只能一人使用，dddd*）

> 💡 **注意**
> **办卡本身免费，请警惕一切说收取开卡费的推销者！**
> **推荐前往官方营业厅（西区大食堂）线下办理以避免被代理额外收费，他们推销成功营业厅会给他们钱而不是给你钱，更不是由你去给他们开卡的钱！**
